Moja konfiguracja systemu

Szczegółowy opis mojego środowiska Linux, terminala Zsh oraz narzędzi developerskich których używam na co dzień

System operacyjny
  • Distro: Ubuntu-based (customized)
  • Shell: Zsh z Oh My Zsh
  • Edytor: VS Code + Vim/Neovim
  • Motyw: Dracula Pro (VS Code), Dracula (terminal)
  • RAM: 32GB DDR4
  • Storage: 1TB NVMe SSD
Terminal & Shell
  • Terminal: Alacritty
  • Shell: Zsh (zcustom config)
  • Klawiatura: Mechanical (Cherry MX Brown)
  • Monitor: Dual 27" 1440p IPS
  • Uruchamianie: Systemd z optymalizacjami
Zsh Configuration

Plugin menedżer: Antibody (lekki i szybki)

Prompt: Powerlevel10k (p10k) z custom segmentami

Historia: 50k w pamięci, 100k na dysku z deduplikacją

Wyszukiwanie: Fzf z integracją historii i plików

Autouzupełnianie: Zsh-autosuggestions + Zsh-completions

Bezpieczeństwo: NO_NOMATCH, GLOB_DOTS, hist ignore all dups

Keybindings & Aliases

Navigation: vim-like bindings (^P, ^N dla historii)

Clipboard: xclip/xsel integracja

Theme: Dynamiczne przełączanie light/dark

AI: Integracja z Ollama (qwen2.5-coder)

Developer Tools
  • Edytory: VS Code (primary), NeoVim (terminal)

  • Languages: JS/TS, Python, Rust, Go, Bash

Przykładowa konfiguracja (.zshrc fragment)
# ==============================
# KXL SYSTEM - ZSH CORE CONFIG
# ==============================

[ -z "$ZSH_VERSION" ] && return

# ==============================
# SAFETY
# ==============================
setopt NO_NOMATCH
setopt GLOB_DOTS

# ==============================
# HISTORY
# ==============================
HISTSIZE=50000
SAVEHIST=100000

setopt APPEND_HISTORY
setopt SHARE_HISTORY
setopt INC_APPEND_HISTORY
setopt HIST_IGNORE_ALL_DUPS
setopt HIST_REDUCE_BLANKS

# ==============================
# COLORS
# ==============================
autoload -Uz colors
colors
setopt PROMPT_SUBST

# ==============================
# PROMPT
# ==============================
PROMPT='%F{cyan} %n%f %F{blue} %m%f %F{yellow} %~%f ${vcs_info_msg_0_} %F{green}❯%f '

# ==============================
# PLUGINS
# ==============================
[ -f /usr/share/zsh-autosuggestions/zsh-autosuggestions.zsh ] && \
  source /usr/share/zsh-autosuggestions/zsh-autosuggestions.zsh

[ -f /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh ] && \
  source /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

# ==============================
# ENVIRONMENT
# ==============================
export OLLAMA_HOST="http://localhost:11434"
export OLLAMA_MODEL="qwen2.5-coder:1.5b"
export EDITOR=nano

# Custom aliases and functions are loaded from ~/.zsh/commands.sh
source ~/.zsh/commands.sh
